* Arduino MEGA
* Keypad 4x4

# Wiring

| keypad 4x4 | pin |
|------|------|
| column 1 | 30 |
| column 2 | 32 |
| column 3 | 34 |
| column 4 | 36 |
| row 1 | 22 |
| row 2 | 24 |
| row 3 | 26 |
| row 4 | 28 |

# How to use it
* Add library to Arduino Libraries folder
* Open example
* Build and upload
